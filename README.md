# mobazha.org

Brand website for [Mobazha](https://mobazha.org) — a decentralized commerce platform for independent sellers.

## Tech Stack

- [Astro](https://astro.build/) v6 — Static Site Generator
- [Tailwind CSS](https://tailwindcss.com/) v4
- [Cloudflare Pages](https://pages.cloudflare.com/) — Hosting

## Pages

| Page | Path | Description |
|------|------|-------------|
| Home | `/` | Hero, Trust Bar, Features, Social Proof, CTA |
| Pricing | `/pricing` | Plan tiers, Founding Seller program |
| For Sellers | `/sellers` | Benefits, competitor comparison |
| Blog | `/blog` | Articles and announcements |
| 404 | `/404` | Custom error page |

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
