What if your AI assistant could set up your online store, import products from Shopify, and write all your product descriptions — just by asking?

That's now possible with **Mobazha Skills**, an open-source plugin that teaches AI coding assistants how to work with Mobazha.

## What are AI Skills?

AI coding assistants like [Claude Code](https://claude.ai/code), [Cursor](https://cursor.com), [Codex](https://openai.com/codex), and others are already helping millions of people write code, manage servers, and automate tasks. Mobazha Skills gives these assistants deep knowledge of the Mobazha platform, so they can guide you through setup, store management, and growth — step by step.

Think of it as a knowledgeable assistant that knows everything about running a Mobazha store.

## What can it do?

### Deploy your store in minutes

Tell your AI: *"Help me set up a self-hosted Mobazha store on my VPS"*

The assistant will walk you through the entire Docker deployment, including domain configuration and TLS setup. It works for native binary installs too — on Linux, macOS, or Windows.

### Import products from other platforms

Tell your AI: *"Import my products from Shopify into Mobazha"*

The assistant knows how to extract product data from Shopify CSV exports, Amazon listings, and Etsy shops — then convert everything into Mobazha's format, including titles, descriptions, images, variants, and pricing.

### Generate product descriptions and marketing copy

Tell your AI: *"Write SEO-optimized descriptions for all my products"*

The assistant can generate product descriptions that convert, store profile copy that builds trust, and even AI image prompts for product photography.

### Manage your store hands-free

Connect your AI to your store via MCP (Model Context Protocol), and it gets direct access to 30+ management tools — products, orders, messages, discounts, and more. Ask it to check sales, update inventory, or respond to customers.

## How to install

One command for each platform:

**OpenClaw** (362K+ stars, the fastest-growing AI agent):

```bash
openclaw plugins install mobazha
```

Also available on [ClawHub](https://clawhub.ai/fengzie/mobazha) — search "mobazha" to browse all 14 skills.

**Claude Code:**

```bash
claude install mobazha/mobazha-skills
```

**Cursor:**

Search "mobazha" in Cursor Settings → Skills, or install from the [Cursor Marketplace](https://cursor.com/marketplace).

**Codex CLI:**

```bash
codex install mobazha/mobazha-skills
```

**OpenCode:**

```bash
opencode plugin add mobazha/mobazha-skills
```

Full installation instructions for all platforms (including Gemini CLI and GitHub Copilot) are in the [GitHub repository](https://github.com/mobazha/mobazha-skills).
